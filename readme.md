# REGISTRO DE PARTIDAS DE BOARDGAMES

## Descrição
Pensando em praticar o desenvolvimento web (front e back) decidi criar este projeto que une dois hobbies meus: programação e jogos de tabuleiro.

O objetivo é simples, criar um site que sirva para registrar as partidas de jogos de tabuleiro que eu jogo com meus amigos. O site será responsável por armazenar as seguintes informações:

- Nome do jogo
- Nome dos jogadores
- Data e hora da partida
- Tempo de partida
- Resultado da partida
- Notas sobre a partida

## Tecnologias

O projeto será desenvolvido com as seguintes tecnologias:

- **Front-end:** React
- **Back-end:** FastAPI
- **Banco de dados:** PostgreSQL

## Estrutura do Projeto

### Front-end

O front-end será responsável por:

- **Login:** Login com e-mail e senha
- **Cadastro:** Cadastro de novos usuários
- **Dashboard:** Dashboard com as últimas partidas jogadas
- **Registro de partidas:** Formulário para registrar novas partidas
- **Histórico de partidas:** Lista de todas as partidas jogadas
- **Perfil:** Perfil do usuário com estatísticas
- **Estatísticas:** Gráficos e estatísticas sobre as partidas jogadas
- **Menu de navegação:** Menu com links para todas as páginas
- **Menu de perfil:** Menu com links para perfil, configurações e logout

O back-end será responsável por:

- **Autenticação:** Autenticação de usuários
- **Registro de partidas:** Registro de partidas
- **Histórico de partidas:** Histórico de partidas
- **Perfil:** Perfil do usuário
- **Estatísticas:** Estatísticas sobre as partidas jogadas
